# Registered Face Dataset

The Streamlit app writes registered face images here.

Expected runtime structure:

```text
dataset/
  STUDENT01_Student_Name/
    STUDENT01_20260515_102000_001.png
    STUDENT01_20260515_102001_002.png
```

Use the app's **Face Registration** section to create these folders from webcam captures or uploaded student photos. Keep real student face images out of public GitHub repositories unless you have consent.
